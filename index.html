Windows 11 VR Setup

Optimize your Windows 11 machine for VR gaming with a single command

powershell -ExecutionPolicy Bypass -Command "iwr -useb https://cederikdotcom.github.io/Hydra-public/setup.ps1 | iex"

Simply copy the command above, paste it into Command Prompt during Windows Audit Mode, and press Enter

What This Script Does

Latest Drivers

Automatically installs the latest GPU, motherboard, and VR device drivers for optimal performance.

Performance Optimization

Configures Windows settings for maximum VR performance, reducing latency and increasing frame rates.

Essential Software

Installs and configures all necessary VR software platforms, utilities, and gaming applications.

How It Works

1

Enter Audit Mode

During Windows setup, press Ctrl+Shift+F3 at the OOBE screen to enter Audit Mode, which allows for clean system configuration before creating a user account.

2

Run the Command

Open Command Prompt in Audit Mode and paste the single-line command. This downloads our setup script and begins execution.

3

Automatic Setup

The script installs package managers (Chocolatey and WinGet), then uses them to install and configure all necessary software.

4

Optimization

Windows settings are automatically tuned for VR performance, including power plans, visual effects, and system configurations.

5

Create User Account

After setup is complete, run Sysprep to exit Audit Mode and create your user account with all optimizations already in place.

Frequently Asked Questions

Is this safe to run?

Yes! The script is open source and available on GitHub. You can review all code before running it. The script doesn't collect any data or make any irreversible changes to your system.

What is included in the setup?

The script installs and configures drivers, VR software platforms (Steam VR, Oculus, etc.), essential utilities, and optimizes Windows for maximum VR performance. It also includes a helper script to create a local user account before leaving Audit Mode.

What VR headsets are supported?

The script supports all major VR headsets including Oculus/Meta Quest, Valve Index, HTC Vive, and Windows Mixed Reality headsets.

How long does setup take?

The entire process typically takes 15-30 minutes, depending on your internet connection speed and system specifications.
